# Customer Shopping Behavior Analysis

## Overview

This project focuses on analyzing customer shopping behavior using Python, SQL, and Power BI. The objective is to uncover customer purchasing patterns, identify key business insights, and present findings through an interactive dashboard and professional report.

The project follows the complete data analytics workflow, including data collection, data cleaning, exploratory data analysis (EDA), SQL analysis, dashboard creation, and business reporting.

---

## Dataset

The dataset contains customer shopping information, including:

* Customer Demographics (Age, Gender, Location)
* Product Categories
* Purchase Amounts
* Shopping Frequency
* Payment Methods
* Discounts and Promotions
* Customer Ratings and Reviews

The dataset was used to analyze customer behavior and generate actionable business insights.

## How to Run

### 1. Clone the Repository

```bash
git clone <repository-link>
cd customer-shopping-analysis
```

### 2. Install Required Libraries

```bash
pip install pandas numpy matplotlib seaborn mysql-connector-python
```

### 3. Run the Jupyter Notebook

```bash
jupyter notebook Customer_shoping_behavior.ipynb
```

### 4. Execute SQL Queries

* Open MySQL Server.
* Import the dataset.
* Run the queries from `Customer_analysis.sql`.

### 5. Open Power BI Dashboard

* Open the `.pbix` file in Power BI Desktop.
* Refresh the data connection if required.
